# AI Models for DeFi Ghost Agents

The DeFi Ghost agents are powered by a combination of large language models (LLMs) and specialized machine learning models, carefully selected to balance reasoning capability, cost, and speed. We leverage the sponsor stack—**OpenAI** and **Venice.ai**—for core language tasks, while smaller fine-tuned models handle domain-specific functions like risk scoring and gas estimation. Integration with **Animoca Minds (Ethoswarm)** provides persistent memory and identity, enabling the agents to learn from past interactions.

---

## Core Language Model: GPT-4 (via OpenAI) or Venice.ai API

- **Usage**: All agents that require natural language understanding and generation (**Supervisor**, **Market Analyst**, **Governance Arbiter**) use GPT-4 or Venice.ai’s equivalent model. The Supervisor interprets user intents (“find best USDC yield on Arbitrum”), the Market Analyst generates biased market commentary, and the Governance Arbiter resolves conflicts based on historical memory.
- **Why**: GPT-4 offers state-of-the-art reasoning and instruction following, crucial for safely handling financial commands. Venice.ai provides a decentralized alternative with comparable performance, aligning with the hackathon’s Web3 ethos.
- **Integration**: Agents call the respective API via LangChain or direct HTTP requests. Prompts are engineered to enforce role-specific behavior (e.g., “You are a bullish market analyst with a focus on DeFi yields. Always cite recent trends.”).
- **Config**: Set `LLM_PROVIDER=openai` or `LLM_PROVIDER=venice` and the corresponding API keys. Model names are configurable via `OPENAI_MODEL` and `VENICE_MODEL` (see `.env.example`).

## Specialized Models for Domain Tasks

| Agent | Model / Approach | Purpose |
|-------|------------------|---------|
| **Opportunity Scout** | XGBoost (lightweight regression) | Trained on historical APY data to predict short-term yield spikes. Runs locally; periodically updated via on-chain data feeds. |
| **Gas & MEV Analyst** | Time-series transformer (e.g. Informer) + heuristic MEV risk classifier | Forecasts gas prices; MEV risk classification fine-tuned on mempool data from Flashbots. |
| **Risk Governor** | Rule-based system + fine-tuned BERT | Classifies protocol risk from audit reports, TVL volatility, and social sentiment. BERT hosted on a private inference endpoint. |
| **Transaction Builder** | No AI model | Uses deterministic smart contract ABIs and encoding libraries (e.g. web3.py) for accuracy and security. |

## Memory and Identity via Ethoswarm

All agents interface with **Ethoswarm’s vector database** for persistent memory. When an agent needs context (e.g., “Have we seen this kind of yield before?”), it queries the memory store using **embeddings** generated by the **sentence-transformer** model `all-MiniLM-L6-v2`. The retrieved memories are injected into the LLM’s prompt as **RAG (Retrieval-Augmented Generation)** context. This allows agents to recall past trades, user preferences, and market patterns without retraining.

- **Embedding model**: `all-MiniLM-L6-v2` (local, via `sentence_transformers`).
- **Storage & search**: Ethoswarm Memory API (`ETHOSWARM_API_KEY`, `ETHOSWARM_MEMORY_STORE_ID`).

## Model Orchestration

- The **Supervisor** acts as the router: it decides which specialist agent to invoke based on the user’s intent and synthesizes final responses using the core LLM (GPT-4 or Venice.ai).
- For tasks like “analyze yield,” the Supervisor broadcasts a structured message; the **Market Analyst** and **Opportunity Scout** run in parallel—LLM for commentary, XGBoost (or equivalent) for numerical prediction.
- Results are aggregated by the Supervisor, which then presents a unified recommendation to the user.

This hybrid approach—combining powerful LLMs with lean specialized models—ensures DeFi Ghost is both intelligent and efficient, capable of running 24/7 while respecting API rate limits and costs. The integration with Animoca Minds (Ethoswarm) ties everything together, giving the system a persistent “soul” that improves over time.
